# 1 Desafio Classificador de Nível de Herói

**O que deve ser Utilizado**
- Variáveis
- Operadores
- Laços de repetição
- Estruturas de decisões


## Obejetivo

crie uma variável para armazenar o nome e a quantidade de
experiência (Xp) de um herói, depois utilize uma estrutura
de decisão para apressentar alguma das mensagens abaixo:

Se XP for menor do que 1.000 = Ferro
Se XP for entre 1.001 e 2.000 = Broze
Se XP for entre 2.001 e 5.000 = Prata
Se XP for entre 6.001 e 7.000 = Ouro
Se XP for entre 7.001 e 8.000 = Platina
Se XP for entre 8.001 e 9.000 = Ascendente
Se XP for entre 9.001 e 10.000 = Imortal
Se XP for maior ou igual a 10.001 = Radiante


## Saída

Ao final deve ser exibir uma mensagem:
"O Héroi de nome **{nome}**" está no nível de **{nivel}**
